About

Hello, my name is Denny, or known as “Doggobot” on the internet, and I am a full stack web developer prioritizing in React and React Native. I have been a proffesional developer for over a year now, and have been actively coding for over 2 years. My preffered stack is React with a docker contained Postgres backend and an Express + knex api. But I am also flexible with many different backend practices, including firebase, sails, and AWS.

The purpose of this website, which was built with Hugo, is to document my many different technical and programming hobbies, as well as some more “leisure” type projects that I have. Many of my hobbies include server infastructure, as well as building keyboards. I am one of the biggest technical nerds you can find!

I will try to update this blog with some fun programming challenges and cool little projects I do occasionally, and just keep this as a more personal fun website. I chose not to build this in something like React or Next as I found for static site generation, Hugo is one of the best offerings.

Happy Coding!
